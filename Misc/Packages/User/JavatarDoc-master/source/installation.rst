Installation
============

Package Control (Recommended)
-----------------------------

Open command palette and type ``Install Package`` then type ``Javatar`` and hit :kbd:`Enter` or :kbd:`Return`.
Package Control will automatically download, install and update for you.

"git" Command
-------------

Open your favourite Terminal application, browse to PACKAGES_PATH and run this command.

::

	git clone git://github.com/spywhere/Javatar.git

Manual Install
--------------

Download .zip file from |J| repository and browse to PACKAGES_PATH, extract .zip file and rename folder to ``Javatar``, restart |st| if you are currently open.

.. note:: PACKAGES_PATH is referred to a folder which can be accessed via the *Preferences > Browse Packages...*
