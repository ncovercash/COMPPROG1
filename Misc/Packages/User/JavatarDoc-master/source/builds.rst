Builds
======

|J| use its own :ref:`build_system` which based on default |st|'s JavaC build settings. Building classes in |J| can be done in several ways...

Project
-------

Building project will let |J| build all Java files in your project.

Package
-------

Building package will let |J| build all Java files in current packages based on current file.

Working
-------

Working is short for working Java files. This will let |J| build all Java files currently open in current window.

Current Class
-------------

Building current class will let |J| build, obviously, current Java file in current view.
